# Mirror-Class-App

미러클라쓰 앱에 대한 코드 및 자료 입니다. 스마트 미러로 부터 필요한 것을 제공 받고 스마트 미러를 제어 하고, 날씨에 따라서 빨래를 할건지 등을 선택할 수 있게 만들 예정입니다.

## 상세 내용

1. 닉네임 정하기
2. 홈 화면
3. ToDo 기능
4. 스마트 기기 제어 (미흡)
5. 날씨 제공기능
